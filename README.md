# AuraGB Programming Language

AuraGB is a custom lightweight programming language designed for simple UI development, structured logic, and system-level interaction. It focuses on a clean syntax model using `agb-` components and command-based execution.

---

## 🚀 What is AuraGB?

AuraGB is a structured programming language designed to:
- Build UI-based applications using simple tags
- Support basic logic operations
- Work with local storage and device interaction (planned)
- Run in lightweight environments like embedded systems or mobile devices

---

## 🎯 Why AuraGB?

AuraGB was created to:
- Reduce complexity of traditional programming languages
- Provide an easy UI-first programming structure
- Support keypad/mobile and embedded device programming
- Combine frontend + logic in a single structured format

---

## ✨ Features

- Simple and readable syntax
- UI component system (`agb-form`, `agb-input`, `agb-button`)
- Basic arithmetic operations
- Local storage support (planned/experimental)
- Lightweight execution model
